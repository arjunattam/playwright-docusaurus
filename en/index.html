Get Started

Test across all modern browsers

Single API to author browser tests for Chromium, Firefox and WebKit.

Use in your preferred language

Use the API in JavaScript/TypeScript, Python, C# and Java.

Automation without trade-offs

Playwright is built to automate the broad and growing set of web browser capabilities used by Single Page Apps and Progressive Web Apps. Scenarios that span multiple page, domains and iframes. Auto-wait for elements to be ready before executing actions (like click, fill)

Integrates with your workflow

One-line installation. Running npm i playwright auto-downloads browser dependencies for your team to be onboarded quickly. Deploy tests to CI. First-party Docker image and GitHub Actions to deploy tests to your preferred CI/CD provider.
